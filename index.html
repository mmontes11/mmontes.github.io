Management platform for places and routes of touristic interest developed as Master's degree final project.

This platform provides an easy way of promoting POIs and routes of touristic interest of your area. You only need to define the area of interest and the type of POIs. After a few minutes, all this data will be available via a responsive web application and an Android app.

The system consisted in the following parts:

  • Java REST service for exposing and managing POIs and routes. It consumes external services: S3(AWS), Google Maps and Facebook
  • PostgreSQL database with PostGIS spatial extension
  • ETL process to obtain POIs and routes from Open Street Map
  • AngularJS responsive web application
  • Android app
